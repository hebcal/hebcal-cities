# @hebcal/cities
Hebcal city database for candle-lighting times.
Includes about 700 world cities.

## Installation
```bash
$ npm install @hebcal/cities
```

## Synopsis
```javascript
import {Location} from '@hebcal/core';
import '@hebcal/cities';

const location = Location.lookup('Athens');
```
